# DATA-SCIENCE-E-Commerce-Sales-Analysis
PROJECT 1-
Mini e-commerce sales analysis project using Python and Pandas. Covers four concepts: Data Cleaning (removing duplicates, fixing missing and negative values), GroupBy (grouping sales by product, customer, month), Sorting (ranking top and bottom performers), and Aggregation (sum, average, count). Includes sample data and console output

E-Commerce Sales Dashboard


Developed an interactive E-Commerce Sales Dashboard in Excel using Pivot Tables, Pivot Charts, Slicers, and KPI Cards. The dashboard provides insights into sales, profit, orders, customer segments, product categories, and regional performance, enabling data-driven business decisions through dynamic visualizations.

Tools Used

- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
